# Atlas Migration Scripts

These scripts were used to migrate Atlas from Airtable → Supabase (June 2026).

- `migrate_atlas.py` — migrated practices and doctors tables
- `migrate_affiliations_fixed.py` — migrated doctor-practice affiliations (fixed version)
- `fix_shortlists.py` — repaired shortlist/favorites data post-migration

Keep these. If you ever need to re-seed or migrate to a new Supabase project, start here.
